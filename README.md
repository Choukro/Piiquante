## 🧞 Commandes

Toutes les commandes sont exécutées depuis la racine du projet, à partir d'un terminal :

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `nodemon server`          | Démarre le serveur local du Front-end            |
| `npm run ng serve`        | Démarre le serveur local du Back-end             |
